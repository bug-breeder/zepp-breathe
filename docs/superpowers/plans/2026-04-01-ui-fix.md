- [ ] **Step 3: Update `chip()` in `src/column.js`**

Find the `chip()` method (currently starts with `chip(text, { selected = false, onPress, variant = 'default', radius = RADIUS.chip, h = 120 } = {})`) and replace it with:

```js
chip(text, { selected = false, onPress, variant = 'default', radius = RADIUS.chip, h = 120, w: chipW } = {}) {
  const w = chipW !== undefined ? Math.min(chipW, this.w) : this.w;
  const x = this.x + Math.floor((this.w - w) / 2);
  const y = this._slot(h, SPACING.chipGap);
  const { normalColor, pressColor, textColor } = this._chipColors(variant, selected);
  return this._create(hmUI.widget.BUTTON, {
    x, y, w, h,
    radius,
    normal_color: normalColor,
    press_color: pressColor,
    text,
    text_size: TYPOGRAPHY.body,
    color: textColor,
    click_func: onPress,
  });
}
```

The only changes from current code: destructure `w: chipW` from options; compute `w` and `x` before `_slot`; use computed `x` and `w` in the widget props instead of `this.x` and `this.w`.
